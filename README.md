# LYPlayer

### 介绍
基于swift3.0，支持高度自定义UI的视频播放器

### 特点
* 手势控制音量、亮度、播放进度
* 横屏状态锁屏
* 横竖屏旋转
* 自动播放

### CocoaPods安装
#### Podfile
```
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'

target 'TargetName' do
pod 'LYPlayer', '~> 0.1'
end
```
运行下面的命令

```
$ pod install
```
